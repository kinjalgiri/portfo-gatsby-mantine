To show my work as a front-end web-developer, I wanted a site that reflects what I can do. Hence I needed more than my previous WordPress blog. I build this website with the Gatsby.js framework. I styled the website with React and Bootstrap and used JavaScript to code dynamic parts (e.g. the tag filter). The Gatsby.js tooling allowed me to compile the homepage into a static website, which is hosted in AWS CloudFront.

**Frameworks**:

* [Bootstrap](https://getbootstrap.com/)
* [Gatsby.js](https://gatsbyjs.org/)
* [React](https://reactjs.org/)

**Techniques**:

* CSS/SCSS
* HTML
* JavaScript
* Markdown (MDX)
* TypeScript

**Hosting**:

* [AWS CloudFront](https://aws.amazon.com/cloudfront/)
* [AWS Lambda](https://aws.amazon.com/lambda/)
